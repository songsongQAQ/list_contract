## 💡 常见问题快速回答

### Q: 需要改动后端接口吗？
A: **不需要！** 后端已支持动态凭证传递。  
👉 详见 [README_COPYTRADING.md](./README_COPYTRADING.md) → "核心答案"

### Q: 怎样启用跟单模式？
A: 打开设置 → 启用"👥 带单模式" → 输入 API → 保存  
👉 详见 [README_COPYTRADING.md](./README_COPYTRADING.md) → "快速开始"

### Q: API Key 泄露了怎么办？
A: 立即在 Binance 官网删除该 API Key，然后重新创建一个新的。  
👉 详见 [README.md](./README.md) → "💰 Binance API 配置" → "安全建议"

### Q: 能同时管理多个账户吗？
A: 可以！通过跟单模式在两个账户之间切换。  
👉 详见 [README_COPYTRADING.md](./README_COPYTRADING.md)

### Q: 怎样在生产环境部署？
A: 使用 Vercel 一键部署，或使用 Docker 自托管。  
👉 详见 [README.md](./README.md) → "📦 构建和部署"
